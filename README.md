# AUTOKEYS SAT V3.2 EASY

Esta versión es de subida sencilla: solo necesitas `index.html` y `supabase_schema.sql`.

## Pasos

1. Ejecuta `supabase_schema.sql` en Supabase SQL Editor.
2. Abre `index.html`.
3. Busca `PEGA_AQUI_TU_ANON_PUBLIC_KEY`.
4. Sustitúyelo por tu Publishable / Anon Public Key.
5. Sube `index.html` a GitHub o reemplaza el anterior.
6. Vercel desplegará automáticamente.
